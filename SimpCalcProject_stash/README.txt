How to use input.py

1. Place the input .txt file on the same folder level as input.py

2. Under the main() function, there is this variable named 'input_file_path'.
   Set its name to the name of the input file placed down earlier

3. Under the main() function, there is this variable named 'output_file_path'.
   Set its name to the name of output file. 

4. Run input.py

5. See the output file on the same folder level.
